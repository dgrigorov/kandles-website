# Story 1.1: Turborepo монорепо + shared packages scaffold

Status: done

## Story

As a developer,
I want a Turborepo monorepo with all shared packages initialized,
so that all apps share types, DB client, UI components, and email templates from a single source of truth.

## Acceptance Criteria

1. **Given** the repo is cloned and Node ≥ v22.12.0 is active **When** `pnpm install` runs **Then** all dependencies install without errors across all workspaces

2. **Given** `turbo.json` defines build pipeline **When** `turbo build` runs **Then** packages build in dependency order: types → env → db → ui → email → apps

3. **Given** workspace is configured **When** `pnpm --filter @kandles/storefront dev` runs **Then** only the storefront app starts (not admin)

4. **Given** all packages are scaffolded **Then** `packages/db`, `packages/types`, `packages/env`, `packages/ui`, `packages/email` each exist with `package.json`, `tsconfig.json`, and `src/index.ts` stub

5. **Given** Node version requirement **Then** `.nvmrc` contains `22` and root `package.json` has `"engines": { "node": ">=22.12.0", "pnpm": ">=9" }`

6. **Given** `apps/storefront` uses Astro v6.4.3 and `apps/admin` uses Next.js v16.2.7 **When** `turbo typecheck` runs **Then** all packages and apps pass TypeScript checks with zero errors

### `turbo.json` pipeline — задължителен dependency ред

```json
{
  "$schema": "https://turbo.build/schema.json",
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": [".next/**", "!.next/cache/**", "dist/**"]
    },
    "typecheck": {
      "dependsOn": ["^build"]
    },
    "lint": {},
    "test": {
      "dependsOn": ["^build"]
    },
    "dev": {
      "cache": false,
      "persistent": true
    }
  }
}
```

`"dependsOn": ["^build"]` = build on dependencies first → packages build преди apps.

### Правилна package.json структура за shared package

```json
{
  "name": "@kandles/db",
  "version": "0.0.1",
  "private": true,
  "exports": {
    ".": {
      "import": "./src/index.ts",
      "require": "./src/index.ts"
    }
  },
  "scripts": {
    "build": "tsc",
    "typecheck": "tsc --noEmit"
  },
  "devDependencies": {
    "typescript": "catalog:"
  }
}
```

Използвай pnpm `catalog:` protocol за споделени dev deps (TypeScript, ESLint) — предотвратява version drift.
